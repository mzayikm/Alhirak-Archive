# Alhirak Archive

Internal archive platform for browsing, searching, and filtering historical articles.

## Tech Stack
- Frontend: Next.js
- Backend: NestJS
- Database: MongoDB

## Features
- Authentication (login/logout)
- Search with ranking
- Filtering (category, author, date)
- Sorting (newest/oldest)
- Pagination
- Article details page
- Multilingual UI (EN/AR)

## Setup

### Backend
```bash
cd backend
npm install
npm run start:dev

### Frontend
```bash
cd frontend
npm install
npm run dev
